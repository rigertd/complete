Documenting Classes

Constructor Function

In JavaScript, you define a class by creating a constructor. The constructor definition is preceded by a comment block that contains information about the constructor function and the class that it instantiates. 

/**
 * Represents an individual employee to be managed in the payroll system.
 * @class
 * @property {string} firstName     - The employee's first name.
 * @property {string} middleInit    - The employee's middle initial.
 * @property {string} lastName      - The employee's last name.
 * @property {string} ssn           - The employee's social security number.
 * @property {Date} dateHired       - The date the employee was hired.
 */
function Employee() {}

The @class tag indicates that this function is meant to be called with the new keyword to create an instance of a class. The @property tag specifies the name and type of a property of the object that is created by the constructor.
